# LAVE Mineral – AI Enabled Mineral Water Platform

## Overview

LAVE Mineral is an AI-enabled web application developed for a mineral water business to provide an intelligent digital platform for product showcasing, ordering, label customization, and secure online payments.

The platform combines modern web technologies with Artificial Intelligence to enhance customer experience through smart automation, interactive UI, and personalized services.



## Features

* Product catalog for multiple bottle types

  * Zen Bottle
  * Round Bottle
  * Pyramid Bottle
  * Square Bottle
  * Peace Bottle

* AI-powered Label Customization

  * Create personalized bottle labels
  * Real-time preview
  * Smart template generation

* AI Chatbot Assistance

  * Instant customer support
  * Product recommendations
  * Query handling automation

* Secure Payment Gateway

  * UPI Payments
  * Google Pay Integration
  * Fast and secure checkout

* Order Management

  * Track orders
  * Payment status monitoring
  * Delivery updates

* Admin Dashboard

  * Product management
  * Order handling
  * Customer records
  * Inventory management

* Customer Feedback System

  * Ratings
  * Reviews
  * Feedback submission

## Installation

Clone repository:

```bash
git clone https://github.com/Khushi2004-oop/AiEnableLaveMineral
```

Move into project:

```bash
cd your-repo
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
